# 📊 Power BI Data Analytics Project — Data Technician Bootcamp

This repository contains my Power BI project completed during Week 2 of the **Data Technician Bootcamp**, where I transformed raw Excel and OData sources into interactive, insight‑driven reports. The project focused on data cleaning, modelling, and visual storytelling using Power Query, DAX, and a wide range of Power BI visualisations.

---

## 🚀 Project Overview

Across several guided labs and practical exercises, I built Power BI reports that combined multiple data sources, applied data transformations, and visualised key business metrics. The work emphasised:

- Preparing and shaping data using **Power Query**  
- Building a semantic model with relationships, calculated columns, and measures  
- Designing interactive reports with slicers, filters, and drill‑downs  
- Communicating insights through clear, intuitive dashboards  

---

## 🧠 Key Skills Demonstrated

### 🔧 Power Query — Data Cleaning & Transformation
- Imported data from Excel workbooks and OData feeds  
- Removed errors, filtered rows, and replaced missing values  
- Split, merged, and reshaped columns  
- Changed data types and applied transformations for modelling  
- Created reusable query steps for automated refresh  

---

### 🧮 DAX — Calculated Columns & Measures
- Built calculated columns for categorisation and grouping  
- Created measures for dynamic aggregations such as:
  - Total Sales  
  - Average Order Value  
  - Year‑over‑Year comparisons  
- Used functions like `CALCULATE`, `SUM`, `DIVIDE`, and time‑intelligence expressions  

---

### 🎛 Interactive Reporting
- Added slicers for fields such as:
  - Date  
  - Product Category  
  - Region  
- Applied page‑level and report‑level filters  
- Enabled drill‑through and cross‑highlighting  
- Designed layouts for clarity and user‑friendly navigation  

---

### 📊 Visualisations Used

- **Bar & Column Charts** — category comparisons  
- **Line Charts** — trends over time  
- **Pie & Donut Charts** — distribution breakdowns  
- **Maps** — geographic sales insights  
- **Tables & Matrices** — detailed reporting  
- **Cards & KPIs** — high‑level performance indicators  

---

## 📈 Project Highlights

### **Excel to Power BI Report**
- Loaded and modelled sales data from Excel  
- Built a multi‑page report with KPIs, trends, and category breakdowns  
- Published to Power BI Service for sharing  

### **Combining Excel + OData**
- Merged two data sources into a unified model  
- Cleaned and transformed both datasets  
- Created visuals showing combined insights across systems  

### **Power BI Labs (Loading & Designing Reports)**
- Practised loading, shaping, and modelling data  
- Designed clean, professional report layouts  
- Published reports (with some admin restrictions noted)

## 📚 Insights Gained

- Power Query is essential for preparing clean, reliable datasets before modelling.  
- DAX enables dynamic, context‑aware calculations that bring reports to life.  
- Slicers and filters make reports interactive and user‑driven.  
- Visual design choices significantly impact how effectively insights are communicated.  
- Combining multiple data sources provides a more complete analytical picture.
