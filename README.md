# 📊 Power BI Business Intelligence & Data Engineering Capabilities

This repository serves as a professional showcase of my end-to-end Power BI capabilities. I specialize in transforming fragmented, raw data into high-performance semantic models and interactive executive reports that drive data-informed decision-making.

---

## 🛠 Core Technical Proficiencies

### **Data Engineering & ETL (Power Query)**
* **Multi-Source Ingestion:** Proficient in extracting data from diverse sources, including **Excel workbooks, OData feeds, and flat files**, into a unified analytical environment.
* **Advanced Data Shaping:** Expertise in cleaning and restructuring data through column splitting, merging, pivoting, and conditional logic.
* **Automated Refresh Logic:** Designing reusable query steps to ensure data integrity and automated reporting pipelines.

### **Semantic Modeling & DAX**
* **Relationship Management:** Building robust star-schema models with active/inactive relationships and cross-filter direction control.
* **Advanced Analytics with DAX:** Writing complex **Data Analysis Expressions** to create dynamic business metrics:
    * **Calculated Columns:** For static categorization and custom grouping.
    * **Measures:** Utilizing `CALCULATE`, `FILTER`, `ALL`, and `DIVIDE` for context-aware aggregations.
    * **Time Intelligence:** Implementation of Year-over-Year (YoY) growth, Year-to-Date (YTD) totals, and rolling averages.

### **Interactive Visual Storytelling**
* **User-Centric Design:** Creating intuitive layouts that prioritize high-level KPIs before diving into granular detail via **Drill-through** and **Cross-highlighting**.
* **Advanced Interactivity:** Implementing synchronized slicers (Date, Region, Category) and report-level filters to allow for multi-dimensional exploration.
* **Visual Selection:** Strategic use of Line Charts for trend analysis, Matrices for detailed financial reporting, and Geographic Maps for spatial insights.
